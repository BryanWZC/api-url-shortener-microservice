API Project: URL Shortener Microservice

User Story:

A microservice that creates a shortened url for any valid site. Enter a valid website below to be provided a short url id.

Short URL Creation

example: POST [project_url]/api/shorturl/new - https://www.google.com

Example Usage:

[this_project_url]/api/shorturl/1

Will Redirect to:

https://www.google.com
